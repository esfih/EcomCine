# tvt-002 — Archived standalone converter history panel

## Business Objective

Preserve the removed standalone converter history/cleanup UI and its related frontend behavior as a feature snapshot before removing it from the active deployment on shared hosting.

## Archive Reason

- The hosted converter now runs on mutualized hosting with limited disk budget.
- The visible history panel encouraged repeated retention of finished artifacts in the UI.
- Active deployment was simplified to a single conversion flow while server-side temp assets are retained for at most 24 hours and then purged automatically.

### Archived JS behavior

```js
var elHistoryList = $("history-list");
var elHistoryEmpty = $("history-empty");
var elPurgeComplete = $("purge-complete");
var recentJobs = [];

function upsertRecentJob(job) {
  var found = false;

  recentJobs = recentJobs.map(function (entry) {
    if (entry.id === job.id) {
      found = true;
      return job;
    }
    return entry;
  });

  if (!found) {
    recentJobs.unshift(job);
  }

  recentJobs.sort(function (left, right) {
    var leftTime = left.updatedAt || 0;
    var rightTime = right.updatedAt || 0;
    return rightTime - leftTime;
  });

  renderHistory();
}

function renderHistory() {
  if (!recentJobs.length) {
    elHistoryList.innerHTML = "";
    hide(elHistoryList);
    show(elHistoryEmpty);
    return;
  }

  hide(elHistoryEmpty);
  show(elHistoryList);
  // Renders item title, status, detail, and WebM/Poster/Delete actions.
}

function refreshJobs() {
  fetch('/api/jobs', { cache: 'no-store' })
    .then(function (response) { return response.json(); })
    .then(function (payload) {
      recentJobs = payload.jobs || [];
      renderHistory();
    });
}

function deleteJob(jobId) {
  fetch('/api/jobs/' + encodeURIComponent(jobId), { method: 'DELETE' })
    .then(function () {
      recentJobs = recentJobs.filter(function (job) {
        return job.id !== jobId;
      });
      renderHistory();
    });
}

function purgeCompleted() {
  fetch('/api/jobs?status=complete', { method: 'DELETE' })
    .then(function () {
      recentJobs = recentJobs.filter(function (job) {
        return job.status !== 'complete';
      });
      renderHistory();
    });
}
```

## Archived Runtime Contract

- UI consumed `GET /api/jobs`
- UI consumed `DELETE /api/jobs?status=complete`
- UI consumed `DELETE /api/jobs/:id`
- UI surfaced download links returned from the existing job payload

## Current Active Behavior After Removal

- No history panel is rendered in the active converter deployment.
- Conversion remains single-session focused: upload, convert, download, convert another.
- Server-side temp job directories are retained for at most 24 hours and then purged automatically.
